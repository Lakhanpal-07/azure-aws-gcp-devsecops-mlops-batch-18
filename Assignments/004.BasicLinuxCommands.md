## Assignment: Basic Linux Directory Navigation Commands

### Commands Covered

* `mkdir`
* `cd`
* `ls`
* `pwd`

---

### Objective

To understand and practice basic Linux directory-related commands using the **Command Line Interface (CLI)** by creating and navigating folders on a Linux system.

---

### Description of Commands

1. **`mkdir` (Make Directory)**
   Creates a new folder.

2. **`cd` (Change Directory)**
   Moves into a directory or goes back.

3. **`ls` (List)**
   Displays files and folders in the current directory.

4. **`pwd` (Present Working Directory)**
   Shows the current directory path.

---

### Steps to Perform

1. **Check your current directory**

   ```bash
   pwd
   ```

2. **List files and folders**

   ```bash
   ls
   ```

3. **Create a new folder**

   ```bash
   mkdir MyWorld
   ```

4. **Verify the folder creation**

   ```bash
   ls
   ```

5. **Move inside the folder**

   ```bash
   cd MyWorld
   ```

6. **Confirm your location**

   ```bash
   pwd
   ```

---

### Expected Outcome

* Folder **MyWorld** is created successfully.
* Student is able to navigate into the folder.
* Student understands basic Linux navigation commands.
* Student gains confidence using Linux CLI on **Killercoda**.
